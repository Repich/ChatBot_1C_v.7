# Матрица трассируемости требований

Матрица связывает каждое требование из
[продуктовых требований](product_requirements.md) с критериями из
[критериев приемки](acceptance_criteria.md) и, где это применимо, со сценариями
из `tests/corpus/user_questions.yaml`.

Ссылки в матрице описывают проверяемое продуктовое поведение. Они не задают
архитектурные компоненты, внутренний путь выполнения или структуру навыков.
Диапазон `Qxxx–Qyyy` включает обе границы.
